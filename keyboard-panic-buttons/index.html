Your Keyboard Panic Button Questions, Answered

How do my responders view incoming alerts?
Alerts activated by end users are received via the 911Cellular Portal, which is a cloud-based application that can be installed on Windows and Mac computers. Once alerts reach the Portal, they can be forwarded to other team members via SMS, phone call, email and other communication methods.
Can details about previous alerts be viewed?
Yes, the 911Cellular Portal contains detailed information about previous activations. This data can be viewed, sorted and downloaded as reports.
Can this software be installed on laptops and desktops?
Yes, the Keyboard Panic Button application can determine the real-time location of the device it is installed on, such as laptop computers.
What happens if someone triggers an alert from outside our facilities?
The Keyboard Panic Button application will recognize when an alert is triggered from outside of your facilities and notify the user to dial 911 for assistance.
Can alerts be triggered from a computer’s lock screen?
Yes, lock screen activation is available for applications running on the Windows operating system.
Can the Keyboard Panic Button application receive and display info on screen?
Yes, administrators may send mass alerts to computers with the Keyboard Panic Button software installed, which will present as a large ‘screen takeover alert’.
What are the system requirements?
Windows: Windows 7 or more recent, must have a minimum .NET Framework of 4.7.3 Mac: MacOS Mojave or more recent
Is this a cloud-hosted application?
Yes, and as a result, no onsite servers or additional hardware is required.
Is there an API?
Yes, the 911Cellular API is accessible at no additional cost to customers.
How long does it take to get this up and running?
Less than a month. After contract execution, all customers are introduced to a dedicated Onboarding Specialist who will guide them through the process of implementation.
What use cases is the Keyboard Panic Button good for?
The Keyboard Panic Button is great for de-escalating tense and potentially violent situations. This can be an aggressive patient, a scenario where weapons are present, suspicious behavior, etc. The application can also assist during medical emergencies, and during critical (but not emergency) situations (the application can receive and display pertinent information regarding system downtime or business operation disruption).
How much does this cost?
An annual license fee is determined by the number of users and the feature set selection. Once we learn more about your environment, our security experts will provide you with a personalized quote.
